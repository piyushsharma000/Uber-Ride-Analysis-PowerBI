# 🚖 Uber Business Intelligence Dashboard (Power BI)

## 📌 Overview
This project is an interactive **Uber Business Intelligence Dashboard** built using **Power BI** to analyze ride bookings, revenue, customer behavior, vehicle performance, and location insights. The dashboard helps visualize key business metrics and supports data-driven decision making through interactive reports.


## 🛠️ Tools & Technologies Used
- Power BI
- Microsoft Excel (Dataset)
- Power Query
- DAX
- Data Modeling


## 📊 Dashboard Features

### 🏠 Home
- Interactive navigation to all dashboard pages.

### 📈 Overview
- Total Completed Bookings
- Lost Bookings
- Revenue Analysis
- Total & Average Distance
- Customer & Driver Ratings
- Monthly/Quarterly Booking Trends
- Revenue by Vehicle Type
- Top Pickup & Drop Locations

### 🚗 Vehicle Analysis
- Vehicle-wise Revenue
- Customer Count
- Completed Bookings
- Monthly Performance Trends
- Vehicle Comparison Table

### 💰 Revenue Analysis
- Monthly & Quarterly Revenue Trends
- Revenue by Payment Method
- Revenue by Vehicle Type
- Top Customers by Revenue

### 👤 Rider Analysis
- Customer Growth Trends
- Customer Payment Method Analysis
- Ride Frequency Analysis
- Customer Details Table

### 📍 Location Analysis
- Total Distance Trends
- Vehicle-wise Distance
- Busy Pickup Locations
- Time Slot Analysis


## 📌 Key Insights
- Track booking performance and lost bookings.
- Analyze revenue trends across different vehicle categories.
- Identify popular payment methods.
- Monitor customer and driver ratings.
- Discover peak pickup locations and busy time slots.
- Compare vehicle performance using interactive visuals.


## 📂 Dataset
- Source: Excel Dataset
- Data includes ride bookings, revenue, vehicle details, customer information, payment methods, ratings, locations, and trip distance.
